Dashboard de Datos con Fetch API

Alumno: Escandon Marcos Carlos David

Materia: Desarrollo Web Profesional

Este proyecto es un Dashboard dinámico que consume datos de la API JSONPlaceholder y utiliza la IA de Gemini para traducciones en tiempo real.

Requisitos previos:

Node.js instalado.

Instrucciones de ejecución

Clona este repositorio o descarga los archivos.

Abre una terminal en la carpeta del proyecto.

Instala las dependencias necesarias:

npm install


Inicia el servidor local:

node server.js


Abre en tu navegador la dirección: http://localhost:3000

Características principales

Renderizado dinámico: Creación de tarjetas de usuario desde el DOM.

Peticiones Asíncronas: Uso de Fetch API con async/await.

IA Integration: Traducción automática de títulos con Gemini Pro.

Filtro Real-time: Buscador dinámico de usuarios.
